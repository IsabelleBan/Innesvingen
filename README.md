# Innesvingen – Custom Shopify Implementation
**Project Type:** Full-scale E-commerce Launch
**Stack:** Shopify (Liquid, JSON), JavaScript

## Project Overview
Development and launch of Innesvingen’s e-commerce platform. The project focused on performance and a highly customized shopping experience tailored for premium golf equipment.

## Technical Scope
* **Bespoke Theme Development:** Wrote custom Liquid and CSS code to extend Shopify's standard capabilities, ensuring a unique brand identity and UX.
* **Performance Engineering:** Optimized theme code and asset loading to ensure fast response times across all devices.

## Key Features
* Custom product filtering systems.
* Seamless migration of complex product variants and inventory data.
* Integration of local payment gateways and shipping providers.
